# CanSatPowerSupply

缶サット用自作マイコンボード向け電源ユニット  
CR2032 または LIR2032 が使用できます  
LIR2032 使用時，外部電源入力に 5V を供給することで 10mA 定電流ダイオードと 3.9V 定電圧ダイオードでの簡易 CCCV 充電が可能です

また，電源電圧監視用 ADC 入力が可能で，GPIO32 ~ GPIO35 を入力として選択可能です
